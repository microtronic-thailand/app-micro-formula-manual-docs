# คู่มือผู้ใช้งาน MicroAccount

ยินดีต้อนรับสู่ระบบบริหารจัดการบัญชี MicroAccount ระบบที่ช่วยให้การจัดการภาษีและบัญชีของ SME เป็นเรื่องง่าย

## สารบัญ
- **การเริ่มต้นใช้งาน**: ตั้งค่าบริษัทและข้อมูลเบื้องต้น
- **งานเอกสาร**: การออกใบแจ้งหนี้ (Invoice) และใบเสนอราคา (Quotation)
- **คลังสินค้า**: การจัดการสต็อกและ QR Code
- **รายงาน**: การดูงบกำไรขาดทุนและภาษี
